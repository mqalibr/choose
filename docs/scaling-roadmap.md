# Scaling Roadmap (5 -> 50 Stores)

## Phase A (MVP, 0-2 ay)

- 5-6 store
- 150-200 məhsul
- 2-4 saatlıq cron
- SEO əsas landing pages
- Manual scraper health check

## Phase B (Growth, 2-4 ay)

- 15-20 store
- scheduler shard-ları
- error-rate monitorinqi
- Telegram kanalına daily summary
- click tracking / source analytics

## Phase C (Scale, 4-6 ay)

- 30-50 store
- worker pool (Railway/Render/VM)
- queue əsaslı scraping (BullMQ/Cloud Tasks)
- alert engine (price drop per user)
- admin panel ilə match correction

## Future Modules

- Telegram bot command API
- Affiliate link routing + conversion metrics
- Store score/rating logic
- Forecast/price trend insights
